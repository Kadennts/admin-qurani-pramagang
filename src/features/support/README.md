## Support Feature Structure

- `dashboard/`
  - `data/`: query statistik support dashboard
  - `hooks/`: state chart dan statistik support dashboard
  - `model/`: type, constants, dan helper chart
  - `ui/`: tampilan support dashboard
- `tickets/`
  - `data/`: query dan mutasi Supabase untuk support tickets
  - `hooks/`: state list, detail, dan create ticket
  - `model/`: type, constants, dan helper tickets
  - `ui/`: tampilan halaman ticket
